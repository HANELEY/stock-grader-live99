# Stock Grader (Render Deployment)
A simple Express-based API that shows stock details using Yahoo Finance.

### Example:
Visit:
`https://your-app-name.onrender.com/stock/SNOW`

Response Example:
{
  "ticker": "SNOW",
  "name": "Snowflake Inc.",
  "exchange": "NYSE",
  "currency": "USD",
  "price": 384.27,
  "marketCap": "125.2B"
}
